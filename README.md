# 🏥 Inventory & Sales Management System

Web application developed to manage inventory, products, and sales for a pharmacy and beauty retail business.

## 🚀 Features
- Product registration and management
- Inventory tracking and organization
- Product search and filtering (by name, code, or category)
- Product visualization with key details (name, price, category, etc.)
- Simple and intuitive user interface

## 🧠 Technologies
- HTML
- CSS
- JavaScript

## 💡 Project Overview
This system was designed to improve product control and organization in a real business environment, allowing efficient management of inventory and product information.
